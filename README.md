#Introdução à Programação Orientada a Objetos em C++

Este repositório contém códigos desenvolvidos durante meu **primeiro contato com Programação Orientada a Objetos (POO) em C++**.  
O objetivo principal foi **aprender e praticar os conceitos fundamentais da POO**, sem foco em padrões avançados ou arquitetura complexa.

---

#Objetivo do repositório

- Compreender a lógica da Programação Orientada a Objetos
- Praticar a criação e o uso de classes em C++
- Entender como os objetos interagem entre si
- Consolidar a base conceitual da POO

Este repositório representa uma **fase inicial do meu aprendizado** e reflete meu nível de conhecimento naquele momento.

---

#Conceitos estudados

Os principais conceitos trabalhados foram:

- Classes e objetos  
- Encapsulamento  
- Construtores e destrutores  
- Métodos e atributos  
- Organização de código em arquivos `.h` e `.cpp`  

---

#Estrutura do projeto

O repositório está organizado de forma simples, contendo exemplos independentes para praticar cada conceito estudado.
